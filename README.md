Library Management - Book issue

A Full Stack Library Management System developed using Spring Boot, React, javascript, MySQL.
The application manages books, members, and book issue/return operations with REST APIs and a responsive frontend interface.

Tech Stack :
Backend - 
Java
Spring Boot
Spring Data JPA
MySQL
Swagger UI
Frontend - 
React 
Vite
CSS

Features : 
Authentication & Roles - 
Member registration
Role-based users
LIBRARIAN
MEMBER

Book Management - 
Add new books
View all books
View available books
Search books by title or author

Member Management -
Register members
View member details
View books issued to a member

Issue & Return Management -
Issue books to members
Return issued books
Automatic book availability update
